# Logivon Global Technology Services - Official Website

A modern, high-performance agency website built with semantic HTML, CSS3, and Vanilla JavaScript. Designed for speed, SEO, and premium aesthetics.

## Project Structure

```
/
├── index.html      # Main landing page
├── css/
│   └── style.css   # Global styles, variables, and animations
├── js/
│   └── script.js   # Scroll animations and interactions
└── README.md       # This file
```

## Features

- **Premium Aesthetic**: Dark mode design with electric blue accents.
- **Smooth Animations**: Custom fade-up logic using `IntersectionObserver`.
- **Responsive**: Fully optimized for Desktop, Tablet, and Mobile.
- **Zero Dependencies**: No heavy frameworks, ensuring maximum performance.

## Deployment Instructions

This project is static-ready and can be deployed instantly on any platform.

### Option 1: Netlify (Recommended)
1. Log in to [Netlify](https://www.netlify.com/).
2. Drag and drop the `Logivon global tech` folder onto the dashboard.
3. Your site is live instantly!

### Option 2: Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` inside the project folder.
3. Follow the prompts to deploy.

### Option 3: GitHub Pages
1. Push this code to a GitHub repository.
2. Go to **Settings > Pages**.
3. Select the `main` branch and `/root` folder.
4. Save.

## Customization

- **Colors**: Edit the `:root` variables in `css/style.css`.
- **Content**: Update text directly in `index.html`.
- **Fonts**: Currently using 'Outfit' via Google Fonts.
